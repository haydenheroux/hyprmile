## What is HyprMile?

HyprMile is an browser application for data-driven drivers to track their driving habits.
It to make collecting fuel statistics and reviewing travel data as seamless as possible. 
Built-in analysis and calculations determine the true cost of travel based on **your** driving habits.
All of the data stays on your device, and entering location data is completely optional.

## Planned Features

- Self-adjusting location graph weights
- Receipt scanning and automatic log generation
- Map service integration to calculate distance/travel times between locations
